Trading EXP calculation based on incendar's exp chart

  • If the xp chart is correct(i trust incendar's chart), the following trade items give this much xp/item:
  • The EXP values are using Value Pack(+30% exp), i might redo the values if the pack expires.

  • The exp values are calculated from trading between the closest nodes,f.eg. velia-loggia farm. To find the closest node, I'm using famme's distance page and their map(can be accessed from the same webpage) The distance bonus is usually 1.56%, which is pretty close to the base xp value. I'm calculating the base value by dividing the xp by 1.0156((100+1.56)/100)and i get the 100%. Of course there is also the value pack's 30%, so instead of dividing by 1.0156 i divide by 1.3156.
