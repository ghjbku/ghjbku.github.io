  • The exp values are calculated from trading between the closest nodes,f.eg. velia-loggia farm.
    To find the closest node, I'm using famme's distance page and their map(can be accessed from the same webpage)
    The distance bonus is usually 1.56%, which is pretty close to the base xp value.
    I'm calculating the base value by dividing the xp by 1.0156((100+1.56)/100)and i get the 100%.
