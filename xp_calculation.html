  • The exp values are calculated from trading between the closest nodes,f.eg. velia-loggia farm. To calculate the closest node, I'm using famme's distance page and their map(can be accessed from the same webpage) The distance bonus is 1.56%, which is pretty close to the base xp value. I'm calculating the base value by dividing it by 1.0156 and i get the 100%, the base value. Of course there is also the value pack's 50%, so instead of dividing by 1.0156 i divide by 1.5156.
